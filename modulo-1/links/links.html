TAGS

href="../index.html" volta para a pasta anterior

href="endereço do site" endereço do site que o link irá abrir

rel="external" indica que é um link externo. Que vai para fora do meu site

rel="next" indica que está pagina é a posterior

rel="prev" indica que está pagina é a anterior ou principal

target="_blank" abre o link em uma nova guia

target="nofollow" indica para o mecanismo de busca que você não da um aval para buscas neste site

target="_self" especifico para abrir o link na própria aba
